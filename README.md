## What is this?

Matrix Construct projects text through a configurable digital rain effect. What started as a simple animation evolved through 69 iterations into a full production system.

**Zero dependencies. Single HTML file. Just open and go.**


## Use Cases

- 🎬 **YouTube intros/outros** — Cinematic text reveals
- 🎧 **DJ/VJ performances** — Music-reactive visuals with MIDI control  
- 📸 **Photo booth overlays** — Matrix effect on webcam feeds
- 🖼️ **Art installations** — Looping ambient displays
- 🎤 **Presentations** — Professional text reveals
- 📱 **Social media** — Built-in screenshot & video capture

## Features

### 5 Reveal Modes
| Mode | Effect |
|------|--------|
| **Fade** | Characters materialize smoothly |
| **Quantum** | Random characters lock in place |
| **Rain Fall** | Text revealed by falling rain collision |
| **CRT Scan** | Retro scanline reveal |
| **Typewriter** | Classic left-to-right typing |

### Audio & Music
- 🔊 Rain ambience with lock sounds
- 🎤 Voice input — speak and watch words manifest
- 🎵 **Music reactive mode** — rain responds to audio:
  - Bass → density + purple bursts
  - Mids → speed variation  
  - Highs → glitch intensity
- 👏 Clap detection to advance cues
- 🥁 BPM sync for beat-matched sequences

### Production Tools
- 📋 Cue stack system (paste script, spacebar advances)
- ▶️ Auto-play with per-cue timing
- 🔁 Loop mode for installations
- 📦 JSON sequence import/export
- 📸 Screenshot (PNG) and 🎥 Video recording (WebM)

### Control Options
- 🎹 **MIDI controller support** (Chrome/Edge)
- 📡 **Remote trigger** from phone/tablet
- ⌨️ Keyboard shortcuts
- 🖱️ Presentation clicker compatible

### Visual Effects
- Density, speed, size, glow controls
- Glitch effect (digital corruption)
- Wind (bidirectional drift)
- CRT lens (scanlines + vignette)
- Grid overlay (rule of thirds)
- Theater mode (fullscreen, hide UI)

### Media Integration
- Image upload with reveal
- Live webcam with **chroma key** (green screen removal)

## Quick Start

### Option 1: Live Demo
👉 **[dabirdwell.github.io/matrix-construct](https://dabirdwell.github.io/matrix-construct/)**

### Option 2: Download
1. Download `index.html`
2. Open in browser (Chrome/Edge recommended)
3. Type text → Press Execute → Watch it manifest

### Option 3: Clone
```bash
git clone https://github.com/dabirdwell/matrix-construct.git
cd matrix-construct
open index.html  # or just double-click the file
```

- **4,400+ lines** of vanilla HTML/CSS/JavaScript
- **Single-file deployment** — no build step, no dependencies
- **PWA support** — installable on mobile and desktop
- **LocalStorage** — presets and unlocked secrets persist

## The Story

This project evolved through 69 versions of human-AI collaboration. Each iteration added features, fixed bugs, and pushed the boundaries of what a "simple animation" could become.

The code was written through conversation — David describing what he wanted, Claude implementing it, back and forth until something genuinely useful emerged. It's a demonstration of what's possible when humans and AI create together as partners rather than tool and user.

*"We believe we are here to create — together."*
