AUDIO · 🔊 🎙️ 🎵

🔊 RAIN SOUND

Ambient rain audio plus lock/click sounds when characters reveal

🎙️ VOICE INPUT

Speak and your words manifest in the rain. Uses browser speech recognition. Button pulses green when active.

🎵 MUSIC REACTIVE

Rain responds to music: bass → density + purple bursts, mids → speed, highs → magenta glitches. Load a file or capture system audio.

CAPTURE · 📷 ⏺

📷 SCREENSHOT

Save current frame as PNG image. Downloads immediately.

⏺ RECORD VIDEO

Record performance as WebM video file. Includes audio if music is playing. Button turns red while recording. Click again to stop and download.

SEQUENCES · 📋 ▶ 🔁 ♫

📋 IMPORT/EXPORT

Save your cues and settings as JSON to share with others. Import sequences by pasting JSON or dragging a .json file.

▶ AUTO-PLAY

Play imported sequence with timing. Each cue advances automatically based on delay values. Button turns green while playing. Shows progress indicator.

🔁 LOOP

When enabled, sequence repeats continuously from the beginning after reaching the end. Great for installations or ambient visuals.

♫ BPM SYNC

Sync cue timing to detected beat. When music reactive is on, cues advance based on "beats" field in JSON instead of "delay". Auto-detects BPM from bass.

TRIGGER · 🎤 📡 🎹

🎤 AUDIO TRIGGER

Clap or any loud sound triggers next cue. Great for hands-free operation when you can't touch the keyboard.

📡 REMOTE

Trigger from another tab or device. Best used with a wireless presentation clicker which sends keyboard events directly.

🎹 MIDI

Trigger cues from any MIDI controller — foot pedals, DJ pads, keyboards. Pads trigger cues, knobs control parameters. Chrome/Edge only (not Safari).

EXAMPLE WORKFLOWS

🎬 YOUTUBE INTRO

1. Type your channel name
2. Set reveal mode to QUANTUM
3. Click ⏺ to start recording
4. Press Space to execute
5. Click ⏺ again to stop
6. Import WebM to your editor

🎤 LIVE PERFORMANCE

1. Paste lyrics in CUE STACK
2. Click 📡 and open on phone
3. Click ⛶ for fullscreen
4. Tap phone to advance cues
Or: Use 🎤 to trigger by clapping

🎵 DJ/VJ SET

1. Click 🎵 → System Audio
2. Share your music app tab
3. Set baseline: density 30, glitch 0
4. Click 🎙️ to add voice
5. Rain reacts to beat + your words

📋 SHARE A SEQUENCE

1. Set up your cues + settings
2. Click 📋 → EXPORT
3. Send .json file to friend
4. They click 📋 → paste → IMPORT
5. Click ▶ to auto-play with timing

📸 SOCIAL MEDIA

1. Type your message
2. Adjust colors/effects
3. Press Space to execute
4. Wait for best frame
5. Click 📷 to capture PNG
6. Post directly to socials

🎭 THEATER PRODUCTION

1. Create sequence JSON with cues + delays
2. Import on show computer
3. Connect to projector
4. Click ▶ to auto-play
Cues sync to your script timing

🎹 MIDI CONTROLLER

1. Click 🎹 to enable MIDI
2. Connect controller/foot pedal
3. Pads/keys trigger cues
4. Knobs control glitch/density/glow
Program changes select presets

🔁 AMBIENT LOOP

1. Create short sequence (3-5 cues)
2. Import and click 🔁 for loop
3. Click ▶ to start
4. Enable 🎵 music reactive
Perfect for installations
